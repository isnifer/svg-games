Grunt-boilerplate
=================

## Blank template for each new project ##

##### Install dependencies #####
```bash
npm install --save-dev
```

##### Run #####
```bash
grunt
```

##### Minify common.js #####
```bash
grunt min
```

##### CSS Sort and Minify (csscomb, cssmin) #####
```bash
grunt comb
```

##### FTP-deploy (before that check your FTP settings: IP, login, pass, etc.) #####
```bash
grunt deploy
```
